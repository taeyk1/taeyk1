AI·백엔드 기반으로 실제 서비스에 연결되는 기능을 만드는 개발자입니다.

---

## Featured Project

### ChoboRunner AI Server

스마트폰 측면 러닝 영상을 기반으로 초보 러너의 자세 지표를 분석하고, 사용자가 이해할 수 있는 피드백으로 연결하는 AI 서버입니다.

- **Role**: AI 파트 설계 및 구현
- **Tech Stack**: Python, MediaPipe Pose, OpenCV, FastAPI
- **Quality & DevOps**: Pytest, Docker, GitHub Actions
- **API Features**: REST API, WebSocket, overlay video generation

#### Core Features

- 러닝 자세 지표 분석: IC hybrid, Trunk Lean, Knee Flexion, Foot Strike
- visibility 기반 품질 판단
- reason code 및 판단 불가 처리
- window averaging 기반 안정화
- 분석 결과 overlay 시각화
- FastAPI 기반 AI 서버 구조화
- Docker 기반 실행 환경 구성
- GitHub Actions 기반 테스트 자동화
